# Iris Dataset Visualization and Exploratory Analysis

## Objective

The objective of this project is to analyze and visualize the Iris dataset using different graphical techniques. The project aims to understand the distribution of features, identify relationships between variables, and compare the different Iris species through visual representations.

## Dataset

Iris Dataset

Total Records: 150

Features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Visualizations Performed

### 1. Histogram

A histogram was created to understand the distribution of Sepal Length values.

### 2. Scatter Plot

A scatter plot was created to analyze the relationship between Sepal Length and Petal Length.

### 3. Bar Chart

A bar chart was used to compare the number of observations across different Iris species.

## Key Findings

* Most flowers have sepal lengths between 5 cm and 6.5 cm.
* A positive relationship exists between sepal length and petal length.
* The dataset is balanced, with 50 samples for each species.

## Conclusion

The Iris dataset was successfully visualized using multiple chart types. The analysis helped identify feature distributions, variable relationships, and species composition. Data visualization proved effective in understanding patterns within the dataset.
